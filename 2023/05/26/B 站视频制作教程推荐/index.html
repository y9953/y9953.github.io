如果你能掌握上述所有知识点，恭喜你已经成为一名大佬了。

继续推荐：

HeyDrones

有着非常多入门的拍摄和剪辑教程，是 B 站有关无人机最专业的频道。

我是于干

如果你很喜欢做一些 Pr 的小效果的话，请务必关注这个 UP 主。话少干货多。

江君是也

江君是一个温柔的男孩子。他有很多 Ae 动画教程和 Pr 教程，而且还有很多剪辑知识科普，我给他充过电。

OtatoB 土豆

制作了很多 Ae 和 Pr 的后期教程。如果你想做 AMV 类型的视频请务必关注这个 UP 主。

贰_33

主业是翻译油管最火的影视 Youtuber ——Peter Mckinnon 的教程。我在他只有两百粉的时候就关注了他，现在他已经有两万粉了。除了 Peter Mckinnon 之外他也会翻译非常多的国外优秀作品和教程。

8K8K

从前主业是翻译 MKBHD 的视频，现在又翻译了很多国外优质摄影、摄像教程，我所知的第一个会给字幕中较难单词短语做注释的 UP 主。更新速度奇快。看他的视频，不仅能学摄影，还能学英语。

电影传送门主页妹

主要翻译一些对经典电影剪辑、调色、声音、镜头调度进行赏析的视频。坦诚的说我们很难通过看这些视频来提高自己的视频质量，但是看一看真的很有意思，对提升自己的鉴赏水平很有帮助。以后看电影不仅觉得好，还知道哪好。

摄瘾研究所

院线级别的器材评测视频，有关滑轨的评测和教程视频堪称 B 站制作水平最佳。但希望大家不要关注这个频道，更新速度极慢，会让人十分痛苦。

有颜值的咸鱼君

他做了相当多的 FCP 教程，如果你想学 FCP 的话可以关注他。

怒豆渣

没有怒豆渣的电影自习室就没有现在百花齐放的影视教程频道。2014 年开始更新的电影自习室依然是制作水平最高，干货最多的影视教程。现在渣哥有了 B 站账号，又把九十多期电影自习室补档了，请大家一定要去看。渣哥虽然 B 站粉丝量不多，但是电影自习室是全网播放破亿的节目。如果你能把这九十多期全看完再看完官网 399 元的电影自习室 Pro，那么前面提到的所有教程都是小菜一碟了。

视频加载失败

以上就是我用于学习的一些频道了。如果你是小白，那么这些足够你看半年的； 如果你是大佬，很抱歉这里没有特别高级的布光、特效、声音设计、动画和调色教程。一方面是因为我水平不够，一方面是因为这类型的教程的确很少。希望我出《如何在 B 站学习做视频 2》时，能够有足够的水平指导大家学习。最后感谢上面提到的所有 UP 主。
